# Distillation Intelligence Platform

## Overview

Distillation Intelligence Platform is a Physics-Informed AI system designed to improve distillation column performance through predictive analytics, anomaly detection, optimization, and digital twin simulation.

The platform combines Chemical Engineering principles with Machine Learning to provide interpretable and actionable process intelligence.

---

## Problem Statement

Distillation columns are among the most energy-intensive units in chemical and petrochemical industries.

Small deviations in operating conditions can lead to:

* Increased energy consumption
* Reduced separation efficiency
* Product quality issues
* Operational instability

This project aims to provide an AI-assisted decision-support system that helps engineers monitor, optimize, and simulate distillation operations.

---

## Key Features

### Physics_SEI Prediction

Predicts a Physics-Informed Separation Efficiency Index using machine learning models trained on process data.

### Anomaly Detection

Detects abnormal operating conditions using Isolation Forest algorithms.

### Process Optimization

Recommends operating conditions associated with high-efficiency operating regimes.

### Trend Analytics

Visualizes historical process behavior and performance indicators.

### Digital Twin Simulator

Allows engineers to perform what-if analyses by changing operating conditions and observing predicted impacts.

### Engineering Reports

Generates downloadable operational reports for decision support.
